== Description ==

Das Widget zeigt Logo und neueste Nachrichten der [Aktion Libero](http://www.aktion-libero.de/). Die Darstellung ist &uuml;ber die CSS-Klasse »aktionlibero« ver&auml;nderbar, Menge und Umfang der Inhalte k&ouml;nnen bestimmt werden.

== Frequently Asked Questions ==

= Worum geht es bei der Aktion Libero? =

Die »Aktion Libero« ist eine Aktion der deutschen Sportblogs gegen Homophobie im Fu&szlig;ball, Aktionsstart ist der 16.11.2011. Die Aktionsseite findest du unter www.aktion-libero.de.

== Installation ==

1. Entpacke `aktion-libero.zip`.
2. Lade den `aktion-libero`-Ordner in dein Verzeichnis `/wp-content/plugins` hoch.
3. Aktiviere das Plugin im 'Plugins'-Men&uuml; der Wordpress-Administration.
4. F&uuml;ge das Widget unter Design->Widgets deiner Sidebar hinzu und bearbeite die Widget-Optionen.
